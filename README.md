# 👥 HR Analytics Dashboard

## 📖 Overview
The **HR Analytics Dashboard** is an interactive **Tableau** project designed to analyze employee data and provide insights into workforce performance, employee attrition, job satisfaction, and demographics. The dashboard helps HR professionals identify trends and make informed decisions through intuitive visualizations.

---

## 🎯 Problem Statement
Employee attrition and workforce management are critical challenges for organizations. This project addresses these challenges by transforming HR data into an interactive dashboard that enables users to monitor key HR metrics, identify attrition patterns, and understand workforce demographics for better decision-making.

---

## 🛠️ Tools & Technologies
- 📊 **Tableau**
- 📄 **Microsoft Excel**
- 📈 **Calculated Fields**
- 🎨 **Dashboard Actions & Filters**
